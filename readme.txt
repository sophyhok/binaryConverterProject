This program is writen in Java language.
I used NetBeans IDE 8.1 to compile and run it.

1. Create a new project of Java Application by open NetBeans
File -> New Projects -> Java Application -> Next -> Finish

2.  Copy the BinaryConverter.java file and paste 
under the Source Packages of your project name.

3. To compile and run the program click Run -> Run Projects

4. On the Output screen, please follow the instructions 
line by line carefully.

I hope this program works well as you expected.
